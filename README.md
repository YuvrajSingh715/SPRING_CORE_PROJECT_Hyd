# Spring Core Project 🌱

A beginner-friendly project to learn the fundamentals of **Spring Core** using **Java**, **Maven**, and **XML-based configuration**.

## 📌 About

This repository contains my hands-on practice while learning Spring Core. Each commit represents a step in my learning journey, covering core concepts with simple examples.

## 🛠️ Tech Stack

* Java
* Spring Core
* Maven
* Eclipse IDE
* XML Configuration

## 📂 Project Structure

```text
SpringCoreDemo
├── src
│   ├── main
│   │   ├── java
│   │   └── resources
│   └── test
├── pom.xml

```

## 📚 Topics Covered

* Spring Core Introduction
* Inversion of Control (IoC)
* Dependency Injection (DI)
* Bean Configuration
* XML Configuration
* Constructor Injection
* Setter Injection
* Bean Scope
* Bean Lifecycle
* Autowiring
* Collection Injection
* Annotation-Based Configuration

> More topics will be added as I continue learning.

## 🚀 Getting Started

### Prerequisites

* JDK 17 or later
* Maven
* Eclipse IDE (or any Java IDE)

### Run the Project

1. Clone the repository

```bash
git clone https://github.com/YuvrajSingh715/SPRING_CORE_PROJECT_Hyd.git
```

2. Open the project in Eclipse.

3. Update Maven dependencies.

4. Run the desired Java class containing the `main()` method.
